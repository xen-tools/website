About Xen

Xen is an open source virtualisation product released by the University of Cambridge.

It allows your machines to run multiple independant Linux systems upon one physical host.

 
Xen Software

xen-shell is a simple console shell for a hosting company to allow customers to control their own Xen instance.

xen-tools is a collection of software which allow you to create new guest domains.

 
About This Site

This site is designed to collect resources useful to people running Xen upon Debian GNU/Linux, and hosts several pieces of software which can be useful to Xen users:

  • rinse - A tool to bootstrap RPM-based distributions.
  • xen-shell allows users to control their own Xen instance without giving up host access.
  • xen-tools makes installing new Xen guests automatic.
