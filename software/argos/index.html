Argos

Argos was a project which replaced Steve's previous attempt at a distributed Xen control panel. Unlike the previous project this control system is designed to allow you to control the Xen instances running upon an arbitary number of Xen hosts.

Project no more exists

Steve has decided to stop the developement of Argos. Please direct all further questions to him.
