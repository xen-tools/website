Xen Tools Stable Releases

The most recent stable version of Xen Tools is 4.2 and you can download it via the following link:

  • xen-tools-4.2.tar.gz - 230k (MD5: d23afd836451cc2f67129c2568c0f586)
  • xen-tools-4.2.tar.gz.asc - 1k (MD5: ff23146d0d12aa067ae562224d49248f; GPG signature which may be validated using Axel's public key).
  • xen-tools-4.1.tar.gz - 124k
  • xen-tools-4.1.tar.gz.asc - 1k (GPG signature which may be validated using Steve's public key).

Binary packages of the project are included in the Debian GNU/Linux Project, in both the Unstable and Testing branches.

I maintain a package for Debian stable upon my personal apt repository.

If you'd like to be informed of new releases please consider subscribing to xen-tools freshmeat project - if you did this it would be appreciated if you would rate the project.
