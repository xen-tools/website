Source Repository

All of the development of the xen-tools software is conducted via a public Mercurial Repository, if you'd like to use the most recent version of the code then please consider installing from there.

You can checkout a copy of the current version of the code by running the following two commands:

hg clone http://xen-tools.repository.steve.org.uk/

Once you have a copy of the project checked out you can keep it up to date by executing "make update".
