Reporting Bugs

If you're using the current packages included as part of the Debian GNU/Linux distribution please report any bugs using the distributions reportbug command.

Reporting your bugs using the Debian tools allows everybody to see and comment upon any open issues:

  • View all open bugs

If you're installing the package from source, or using a non-Debian distribution then please report your bug to the xen-tools mailing list instead.

This will allow other users to comment and confirm the bug, or suggest a work-around before it can be fixed.
