Mailing Lists

There are two mailing lists setup which might be interesting to users of xen-tools:

xen-tools-discuss

This is a general purpose mailing list, for reporting bugs, asking questsion or making suggestions.

xen-tools-commits

This is is a developer-orientated mailing list, which receives copies of each commit applied to the source repository.
