Source Access

The xen-shell development occurs in a publically accessible mercurial repository from which you are welcome to follow ongoing development.

If you wish to following the development, or contribute patches, then this is the preferred thing to patch against:

  • xen-shell repository

To pull the most current version, to track it, you would run something like this:

hg clone http://xen-shell.repository.steve.org.uk/
