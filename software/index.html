Xen Software

From here you can learn about and download various pieces of software:

argos

argos is a distributed control panel, and monitoring, system designed to allow a network administrator to easily manage the Xen guest instances running upon an arbitary number of Xen hosts.

The code is still in progress, but already there the core is stable and usable.

rinse

rinse is a simple utility which is designed to install RPM-based distributions into a directory, in a similar fashion to the Debian debootstrap tool.

xen-shell

xen-shell is a simple console based shell which can be used to allow users to control a single Xen instance without giving them access to the Xen host system in any other way.

It would be ideal for a Xen hosting co-op, or company.

xen-tools

xen-tools is a small package of scripts which allow you to easily create new Xen guest domains.

You can choose to create new instances of:

  • Debian Sarge
  • Debian Etch
  • Debian Sid
  • CentOS4

Each new instance will be created with a correctly configured network, and an installation of OpenSSH.
