Source Access

The rinse development occurs in a publically accessible Mercurial repository, so you are welcome to download releases from there.

If you wish to following the development, or contribute patches, then this is the preferred thing to patch against:

  • Rinse Mercurial repository

You can checkout a copy of the current version of the code by running the following two commands:

hg clone http://rinse.repository.steve.org.uk/

Once you have a copy of the project checked out you can keep it up to date by executing "make update".
